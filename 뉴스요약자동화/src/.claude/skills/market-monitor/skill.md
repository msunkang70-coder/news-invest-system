# 시장지표 모니터링

10분 주기로 14개 시장지표를 수집하고 임계값 돌파 시 자동 알림.

## 실행

```bash
python src/main.py --indicators-only
```

| 카테고리 | 티커 | 임계값 |
|---------|------|--------|
| 변동성 | VIX | ≥25/30/40 |
| 환율 | 원달러 | ≥1400/1450 |
| 환율 | DXY | ≥105/107 |

| 원유 | Brent | 일변동≥5% |

| 야간선물 | KOSPI200N | 변동≥1.5% |
| 심리 | Crypto F&G | ≤15/≥85 |
