# 지정학 분석

전쟁·제재·분쟁 뉴스를 구조화 분석하여 시장 영향도를 정량화.

## 에스컬레이션 5단계

- **L1 긴장** (×1.0): 외교 갈등, 성명 발표
- **L2 긴장 고조** (×1.2): 제재 발동, 군병력 증강
- **L3 무력 시위** (×1.5): 미사일 시험, 영공 침범
- **L4 무력 충돌** (×2.0): 실제 교전, 인명 피해
- **L5 전면 위기** (×3.0): 전면전, 핵 위협, 해상 봉쇄

## 핫스팟 지역 (5곳)

1. **중동** — 유가, 안전자산, 항공/해운
2. **대만해협** — 반도체, 기술주, 안전자산
3. **한반도** — 코스피, 원화, 방산
4. **우크라이나** — 에너지, 유럽주, 방산
5. **남중국해** — 해상운송, 물류

## 영향 체인 5대

각 지정학 이벤트가 어떤 자산에 순차적으로 영향을 미치는지 추적.
L3 이상이면 자동 이메일 알림.
