## 1. Executive Summary

개인 투자자가 하루 수백 건의 금융 뉴스·시장지표·지정학 이벤트 속에서 중요한 투자 시그널을 놓치는 문제를 해결한다. 42개 소스에서 뉴스와 시장지표를 실시간 수집하고, LLM 기반 BULL/BEAR 방향성 판단 + 지정학 에스컬레이션 5단계 분류 + 교차 자산 영향 체인 분석을 수행하여, 긴급도에 따라 이메일/Telegram으로 자동 알림한다. 기존 뉴스 파이프라인(v1/v2)을 통합하고, VIX·환율·유가·야간선물 등 시장지표 임계값 모니터링까지 확장하여 투자 의사결정 시간을 90% 단축한다.

## 2. 문제 정의

### As-Is (현재)

| 문제 | 영향 | 정량적 수치 |
|------|------|-------------|
| 뉴스 과부하: 하루 수백 건 중 투자 관련 5~10%만 유효 | 중요 뉴스 누락, 노이즈에 의한 오판 | 일일 뉴스 확인 1~2시간, 누락률 30%+ |
| 시장지표 수치 변동 지연 감지 | VIX 급등, 환율 급변 등 30분~1시간 후 뉴스로 인지 | 뉴스 기반 감지 지연: 평균 45분 |
| 지정학 이벤트 무차별 처리 | 북한 훈련과 이란 교전이 동일 가중치 | 지정학 뉴스 분류 정확도: 0% (미분류) |
| 야간선물 미커버 | 장후 글로벌 이벤트 대응 불가 | 야간(18:00~06:00) 투자 정보: 0건 |
| 이메일 알림 부재 | 텔레그램만 지원 (v1) | 이메일 알림: 0건/일 |

### To-Be (목표)

|------|------|------|--------|

| 중요 뉴스 누락률 | 30%+ | 5% 이하 | 83% 개선 |
| 시장지표 감지 지연 | 45분 (뉴스 대기) | 10분 (직접 수집) | 77% 단축 |

## 4. 핵심 기능 명세

### P0 (필수) — Phase 1~2

| 기능 | 사용자 스토리 | 수용 기준 |
|------|-------------|-----------|
| **F1. RSS 뉴스 수집** | 투자자로서 11개 RSS 소스에서 5분 주기로 뉴스를 자동 수집하고 싶다 | AC1: 11개 피드 정상 파싱, AC2: 5분 주기 동작, AC3: 중복 제거 동작 |
| **F2. 3-Tier 키워드 필터** | 투자자로서 STRONG/MEDIUM/WEAK 등급으로 뉴스를 자동 분류하고 싶다 | AC1: 75개+ 키워드 매칭, AC2: 오탐율 15% 이하 |
| **F3. 다차원 영향도 스코어링** | 투자자로서 뉴스의 긴급도×파급범위×확실성을 1~10 점수로 보고 싶다 | AC1: 3축 독립 계산, AC2: sigmoid 정규분포, AC3: 임계값(5.0) 필터 |
| **F4. LLM 요약 + BULL/BEAR 판단** | 투자자로서 뉴스 요약과 방향성을 자동으로 받고 싶다 | AC1: Gemini 2.5 Flash 연동, AC2: NEUTRAL 없음, AC3: fallback 동작 |
| **F5. Gmail 이메일 알림** | 투자자로서 긴급 뉴스를 이메일로 즉시 받고 싶다 | AC1: OAuth2 인증, AC2: HTML 템플릿, AC3: 전송 성공률 99% |
| **F6. 알림 룰 엔진** | 투자자로서 조건별로 다른 알림을 받고 싶다 | AC1: 5종 뉴스 룰 동작, AC2: 쿨다운, AC3: 배치 큐 |

### P1 (중요) — Phase 3~5

| 기능 | 사용자 스토리 | 수용 기준 |
|------|-------------|-----------|
| **F7. 시장지표 실시간 수집** | 투자자로서 VIX/환율/유가/국채 수치를 10분 주기로 모니터링하고 싶다 | AC1: yfinance 8개 티커, AC2: FDR 3개, AC3: 임계값 검사 |
| **F8. 야간선물 모니터링** | 투자자로서 야간선물 변동을 알림받고 익일 전망을 보고 싶다 | AC1: KIS API 연동, AC2: 18:00~06:00 15분 주기, AC3: 익일 전망 생성 |
| **F9. 지정학 에스컬레이션 분류** | 분석가로서 지정학 뉴스를 L1~L5 5단계로 분류하고 싶다 | AC1: 5단계 키워드 매칭, AC2: 지역 태깅, AC3: Impact 승수 적용 |
| **F10. 교차 자산 영향 체인** | 분석가로서 유가→인플레→금리→주가 등 연쇄 영향을 추적하고 싶다 | AC1: 5대 체인 템플릿, AC2: 2차 파급 종목 태깅 |

| **F12. 경제지표 서프라이즈** | 분석가로서 CPI 등 경제지표의 예상 대비 서프라이즈를 알고 싶다 | AC1: 실제 vs 예상 계산, AC2: 서프라이즈 등급 |

| KIS Open API | 야간선물 (한투) | 확인 필요 (계좌) |

## 11. Out of Scope

- **자동 매매 (Auto-Trading):** 시그널 생성까지만, 실제 주문 실행은 제외
- **유료 뉴스 크롤링:** 저작권 이슈로 공개 RSS/API만 사용
- **실시간 호가 스트리밍:** WebSocket 기반 실시간 시세는 제외 (10분 폴링)
- **모바일 앱:** 이메일/텔레그램 알림으로 대체
- **다중 사용자:** 단일 사용자 전제, 인증/권한 관리 제외
- **다국어 (한/영 외):** 일본어, 중국어 뉴스 미지원
