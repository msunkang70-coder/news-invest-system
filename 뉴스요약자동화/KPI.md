## 1. 핵심 KPI (North Star + 주요 지표)

### North Star Metric

| 항목 | 내용 |
|------|------|
| 지표명 | 투자 의사결정 시간 단축률 |
| 정의 | 일일 뉴스 파악 + 투자 판단에 소요되는 총 시간의 감소 비율 |
| 산식 | (기존 소요시간 - 현재 소요시간) / 기존 소요시간 × 100 |
| 현재값 | 0% (시스템 미가동) |
| 목표값 | 90% (90분 → 10분) |
| 측정 주기 | 주간 |

### 주요 KPI

| KPI | 정의 | 산식 | 현재 | 목표 | 주기 |
|-----|------|------|------|------|------|
| 방향성 정확도 | BULL/BEAR 판단 정확률 | 정확 건수 / 검증 전체 | - | 75% | 월간 |
| 알림 지연시간 | 이벤트 → 알림 도달 | avg(수신시각 - 발생시각) | - | < 2분 | 일간 |
| 이메일 전송 성공률 | 발송 성공 비율 | 성공 / 시도 × 100 | - | 99% | 일간 |
| 중요 뉴스 포착률 | 주요 이벤트 중 시스템이 감지한 비율 | 감지 / 전체 이벤트 × 100 | - | 95% | 주간 |
